# UserAgent

Represents information of the client side.

## Schema

```text
type UserAgent {
  platform : Platform!
  version : string!
}
```

## Params

| Name | Type | Description | Note |
| --- | --- | --- | --- |
| platform | Platform! | Represents the device type of the client. | - |
| version | String! | Represents the OS version. | - |
