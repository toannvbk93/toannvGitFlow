# Platform

Represents the type of devices on the client side which is using the API.

## Schema

```text
enum  Platform {
  iPhone
  Android
  Browser
}
```

## Params

| Name | Description | Note |
| --- | --- | --- |
| iPhone | Represents iPhone app. | - |
| Android | Represents Android app. | - |
| Browser | Represents WEB browser. | - |
